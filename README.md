# 📊 Sales Analytics Dashboard (Excel)

## 📌 Overview

This project showcases an interactive **Sales Analytics Dashboard** built using Microsoft Excel. It provides insights into sales performance, customer behavior, and operational metrics using dynamic charts and slicers.

---

## 🚀 Key Features

* 📅 **Orders vs Sales Analysis**

  * Monthly comparison of order count and sales amount

* 🚚 **Delivery Status Breakdown**

  * Delivered, Returned, Refunded, and Cancelled orders

* 👥 **Customer Insights**

  * Gender-based sales trends
  * Age group analysis (Adult, Senior, Teenager)

* 🛒 **Order Channel Distribution**

  * Amazon, Flipkart, Myntra, Ajio, Meesho, etc.

* 🧵 **Category Filtering**

  * Blouse, Bottom, Ethnic Dress, etc.

* 🗺️ **Regional Insights**

  * Top-performing states by sales

* 🎛️ **Interactive Slicers**

  * Month, Category, Channel filters

## 📂 Dataset

The dataset includes:

* Order details
* Sales amount
* Customer demographics (Gender, Age)
* Order channels
* Delivery status
* State-wise sales data

---

## 💡 Key Insights

* Majority of orders are successfully delivered (~90%+)
* Amazon contributes the highest share of sales
* Adult women are the top purchasing group
* Maharashtra and Karnataka are top-performing states
* Sales fluctuate across months with noticeable seasonal trends

---

## ⚙️ How to Use

1. Download the Excel file from the `dashboard/` folder
2. Open in Microsoft Excel (2016 or later recommended)
3. Use slicers to interact with the dashboard
